# Data Directory

This folder contains JSON data files fetched by GitHub Actions.

When no JSON files exist, the dashboard falls back to representative data built into `app.jsx`.

## Files (populated automatically)

- `wasde.json` — WASDE balance sheets (monthly, ~12th of each month)

## Manual refresh

Go to **Actions** tab → **Fetch WASDE Data** → **Run workflow** to trigger a manual fetch.
